# 一道Javascript考察基本功

考察点：函数、类、原型链、运算符优先级、作用域、变量提升

function Foo() {
    getName = function() {
        console.log(1)
    }
    return this
}

Foo.getName = function() {
    console.log(2)
}
Foo.prototype.getName = function() {
    console.log(3)
}
var getName = function() {
    console.log(4)
}
function getName() {
    console.log(5)
}

Foo.getName()           // 2
getName()               // 4
Foo().getName()         // 1
getName()               // 1
new Foo.getName()       // 2
new Foo().getName()     // 3
new new Foo().getName() // 3

解析：

  1. Foo.getName()
    直接取Foo对象的getName属性执行为2
  2. getName()
    考察变量提升，先声明getName变量，再声明getName函数，再执行getName的函数赋值，实际执行的函数如下
    var getName
function getName() {
    console.log(5)
}
getName = function() {
    console.log(4)

  3. Foo().getName()
    先执行Foo()，执行过程中给全局getName重新赋值，然后返回了this对象window，所以最终调用的是window.getName()，故执行结果为1
  4. getName()
    同3的最后一步window.getName()，结果为1
  5. new Foo.getName()
    先执行Foo.getName()，后实例化，执行结果为2
  6. new Foo().getName()
    先实例化new Foo()，后执行getName()方法，结果为3
  7. new new Foo().getName()
    先实例化new Foo()拿到结果Instance，后实例化new Instance.getName()，结果为3
